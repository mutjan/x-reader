# 量子位风格高阅读高转发选题规则 v2.0

## 一、核心选题原则

### 1.1 黄金选题公式
```
高传播选题 = 热点人物/公司 + 突破性事件 + 情绪共鸣点 + 实用价值
```

### 1.2 选题优先级排序
1. **S级（必报）**：AI大模型重大发布、科技巨头CEO重大动态、Nature/Science顶刊突破
2. **A级（优先）**：开源爆款项目、颠覆性技术、人物励志故事、行业重磅人事变动
3. **B级（可选）**：产品评测、技术解析、行业分析

## 二、高传播选题类型

### 2.1 AI大模型/ChatGPT类（平均阅读9.6万+）
**选题特征**：
- 大模型重大版本发布（GPT-4、Claude、Gemini等）
- 多模态能力突破（视频生成、图像生成）
- AI Agent/自动化工具突破
- 开源大模型重大进展

**标题公式**：
- `「产品名」+ 核心能力`
- `「数字」+ 分钟/秒 + 实现 + 震撼效果`
- `「公司」+ 发布 + 「首个/首次」+ 突破性产品`

**案例**：
- GPT-4发布！ChatGPT大升级！太太太太强了！
- 效果炸裂！OpenAI首个视频生成模型发布，1分钟流畅高清
- ChatGPT王炸更新！能联网获取新知识，可与5000+个应用交互

### 2.2 科技巨头动态类（平均阅读11万+）
**选题特征**：
- 马斯克/SpaceX/特斯拉重大动态
- 微软、谷歌、苹果重大产品发布
- 字节、阿里、百度大模型进展
- 科技巨头CEO言论/争议

**标题公式**：
- `「刚刚/突发」+ 马斯克/公司 + 创造历史/创造人类新壮举`
- `「公司」+ 深夜炸场/突然宣布 + 重大决定`
- `「公司」+ 曝光/揭秘 + 内部重大事件`

**案例**：
- 刚刚，马斯克创造人类航天新壮举！空中炸毁火箭，然后成功实现载人舱逃逸
- 谷歌研究员走火入魔事件曝光：认为AI已具备人格，被罚带薪休假
- GPT-4接入Office全家桶！Excel到PPT动嘴就能做，微软：重新发明生产力

### 2.3 学术/科研突破类（平均阅读10.9万+）
**选题特征**：
- Nature/Science/顶刊发表
- 数学/物理重大理论突破
- 中国学者/华人学者重大突破

- `「数字」+ 个/位 + 研究者 + 颠覆了 + 领域常识`
- `「清华/北大/MIT」+ 研究者 + 开发/实现 + 突破性成果`

**案例**：

### 2.4 人物故事/励志类（平均阅读16.9万+，教育类最高）

## 三、标题创作黄金法则

### 3.1 高传播标题元素（按重要性排序）
1. **感叹号**（53.9%的高传播文章使用）- 增强情绪感染力
2. **数字** - 具体化、可信度高
3. **"网友"引用**（11.8%）- 增加社交证明
4. **"首次/首个"** - 强调突破性
5. **"刚刚/突发"** - 强调时效性
6. **"王炸"** - 强调重要性
7. **顶刊背书**（Nature/Science）- 增加权威性

### 3.2 标题结构模板

**模板1：情绪冲击型**
```
[情绪词]！[产品/人物] + [动作] + [震撼效果]，网友：[评论]
```
例：效果炸裂！OpenAI首个视频生成模型发布，1分钟流畅高清，网友：整个行业RIP

**模板2：悬念揭秘型**
```
[数字/身份] + [颠覆性动作]，[权威人物]：我开始压根不相信
```
例：3个搞物理的颠覆了数学常识，数学天才陶哲轩：我开始压根不相信

**模板3：励志逆袭型**
```
他曾经[低谷经历]，如今[辉煌成就]（[转折细节]）
```
例：他曾经复读才考上三本，如今让华为开出201万年薪（其实还拒绝了360万offer）

**模板4：时效热点型**
```
刚刚/突发，[人物] + [创造历史/重大动作] + [具体成就]
```
例：刚刚，马斯克创造人类航天新壮举！空中炸毁火箭，然后成功实现载人舱逃逸

**模板5：争议爆料型**
```
[公司/机构] + [事件] + 曝光：[震撼内容]，网友：[反应]
```
例：谷歌研究员走火入魔事件曝光：认为AI已具备人格，被罚带薪休假，聊天记录让网友San值狂掉

### 3.3 标题禁用词
- 过度夸张的形容词（"最"、"第一"等，除非有数据支撑）
- 生僻技术术语（除非面向专业读者）
- 负面导向词汇（"失败"、"崩溃"等，除非是重大新闻）

## 四、选题处理规则

### 4.0 多源聚合规则（优先级最高）
当多个不同消息源在同一时间窗口内报道同一事件时，优先选择该事件并聚合为一个条目：
- 将所有来源链接合并到 `sourceLinks` 字段
- `sources` 字段标记来源数量
- 被多源报道的事件自动获得额外加分（每多一个来源 +5 分，最高 +20 分）
- 合并后的条目取最高评分

### 4.1 标题生成规则
- **标题中不带类型标签**：不使用 `【科研】`、`【新品】` 等前缀，类型信息放在独立的 `type` 字段
- **标题必须是量子位风格的纯中文标题**，参考第三章的标题模板
- **标题中不能包含图片**：标题必须是纯文本，不能有 HTML `<img>` 标签或图片链接
- 标题风格参考：
  - ✅ `效果炸裂！OpenAI首个视频生成模型发布，1分钟流畅高清`
  - ✅ `刚刚，马斯克创造人类航天新壮举！空中炸毁火箭`
  - ❌ `【新品】OpenAI：OpenAI 推出 video model——正式发布`
  - ❌ `【科研】Google：researchers 发表 in 《自然》——学术成果发布`

### 4.2 摘要生成规则
- **优先获取原文内容**：通过 URL 抓取文章正文（使用 `curl` 获取页面内容，提取正文文本）
- **基于正文生成一句话总结**：用一句简洁的中文概括文章核心内容
- **降级策略**：如果无法获取原文（网络超时、反爬等），摘要使用 `点击链接查看详情` 代替
- **摘要中不能包含图片**：摘要必须是纯文本，不能有任何 HTML 标签（包括 `<img>`、`<br>` 等）
- 摘要长度控制在 50-100 字

## 八、自动化选题评分标准

### 8.1 评分维度（满分100分）

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 时效性 | 20% | 24小时内发布（20分）、48小时内（15分）、一周内（10分） |
| 人物热度 | 20% | 马斯克/奥特曼级（20分）、大厂CEO级（15分）、知名学者（10分） |
| 突破性 | 20% | 行业里程碑（20分）、重大更新（15分）、常规进展（10分） |
| 情绪价值 | 20% | 强烈共鸣（20分）、励志/争议（15分）、普通（10分） |
| 实用性 | 20% | 可直接使用（20分）、有参考价值（15分）、纯资讯（10分） |

### 8.2 自动过滤规则
- 评分低于60分：不采纳
- 评分60-75分：B级选题，可选
- 评分75-90分：A级选题，优先
- 评分90分以上：S级选题，必报
