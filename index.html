📖 使用说明

由于 X API 的限制和 CORS 策略，纯前端网页无法直接访问 X 数据。以下是几种可行的方案：

方案 1：使用 RSS 订阅（推荐）

  1. 访问 rss.app 或类似服务
  2. 输入你的 X 列表 URL，生成 RSS 链接
  3. 使用 RSS 阅读器订阅
访问 RSS.app

方案 2：使用 Nitter 镜像

  1. Nitter 是 X 的隐私友好镜像
  2. 访问 nitter.net
  3. 搜索你的列表或用户

方案 3：本地运行脚本

在你的本地机器上运行 Python 脚本获取数据：

# 安装依赖
pip install requests

# 运行脚本
python3 fetch_x_list.py

💡 获取 X Bearer Token

  1. 登录 X.com
  2. 按 F12 打开开发者工具 → Network 标签
  3. 刷新页面，找到任意 API 请求
  4. 在 Headers 中找到 Authorization: Bearer xxx
  5. 复制 Bearer 后面的字符串
注意：Bearer Token 相当于你的登录凭证，不要分享给他人。建议定期更换。
